# 灵溪垂钓

这是洞天福地的静态小游戏源码留档，只保留单一玩法本体和修仙服务对接能力。

## 边界

- 玩法固定为 `灵溪垂钓`，不做多模式、难度选择、排行、成就、账号、支付或本地存档。
- 前端只负责游戏过程、展示兑换码和基础奖励预览；奖励由修仙服务端裁定，前端不能直接发奖。
- 鱼群生成可以随机，但结算口径固定：只提交本局分数、钓获记录和洞天单局凭证。
- 源码可用 `npm run build` 重新构建，产物同步到 `xiuxianserver/static/dongtian/lingxi-fishing/`。

## 命令

```bash
npm install
npm run build
```
