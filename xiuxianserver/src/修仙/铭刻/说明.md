# 铭刻组件说明

铭刻负责个性化显示名，只改变外观文本，不改变属性、价格和战斗结算。

## 命令

```text
铭刻
铭刻之羽
铭刻固定装备 装备位 新名字
铭刻武器 武器ID 新名字
铭刻附魔 武器ID 附魔序号 新名字
```

也可以用统一入口：

```text
铭刻 固定装备 头部 青云冠
铭刻 武器 武器#12 青云剑
铭刻 附魔 武器#12 1 青云破
```

## 规则

- 每次成功铭刻消耗 1 个铭刻之羽。
- 铭刻之羽只能由岁时情劫首领产出，不进入普通探险和异界虫洞奖励。
- 铭刻之羽按实例保存，每一枚都有来源首领、标题和文案。
- 发送 `铭刻之羽` 可以查看未使用遗羽和对应文案。
- 铭刻成功后，该枚铭刻之羽直接删除，来源与故事随之消散。
- 铭刻后的对象只保留新名字，不保留来源记录。
- 新名字最长 12 个字符，不能包含空白字符。
- 固定装备仍按原装备位定位，例如：头部、护甲。
- 武器仍按实例 ID 定位，例如：武器#12。
- 附魔按武器 ID 和附魔序号定位，序号在武器列表里查看。
- 铭刻可以覆盖旧名字，每次覆盖都重新消耗材料。
- 展示时保留原名，例如：青云剑（青岚短剑）、青云破（风刃书）。

## 扩展

- 铭刻不改全局定义表，只改玩家自己的实例显示名。
- 新增可铭刻对象时，优先给目标实例表加 `custom_name`。
- 如果目标没有实例表，可以单独建 `目标_id + custom_name` 的映射表。
