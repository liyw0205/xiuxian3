# 商场组件说明

商场负责跑商、地点导航和特殊收购。

## 命令

```text
商场
商场列表
商场详情 地点名
商场市价 商品名
商场购买 商品名 数量
商场出售 商品名 数量
商场自动出售
商场推荐
商场记录
商场限制
商场奖励
特殊收购
特殊出售 物品名 数量
导航 地点名
导航 x y
```

## 规则

- 跑商、导航、商场买卖和特殊出售由“行商化身”处理。
- 行商化身只负责经济行动，不参与首领、虫洞和玩家对战。
- 本体探险时仍可跑商，探险结果按开始探险时的地点预计算。
- 普通跑商商品进入背包，占负重。
- 商场购买只能买当前位置的本地特产，不能跨地点远程购买。
- 特殊收购地点不能购买普通跑商商品。
- 恢复类、宝石、技能书和武器不参与跑商。
- 同地点买入后 4 小时内不能原地出售。
- 普通跑商最高出售参考价按最近买入价的 145% 封顶。
- 每日 04:00 重置跑商奖励；当天完成普通跑商出售后可领取一次额外源石。
- 跑商奖励只统计普通出售，不统计特殊收购。
- 推荐路线按每负重净利润排序，适合背包有限时选货。
- 特殊出售必须先导航到特殊收购地点，只收该地点配置的怪物战利品。
- 查看市价、详情和推荐只计算价格，不写入交易价格缓存。

## 扩展

- 新增跑商地点：改 `TRADE_LOCATIONS`，每个地点至少给出特产。
- 新增特产价格：改 `TRADE_ITEM_DEFS`，缺少定价会在数据库初始化时报错。
- 调整地点偏好：改 `TRADE_LOCATION_DEMANDS`。
- 新增特殊收购：改 `SPECIAL_BUYERS`，并给出坐标、收购物品和倍率。
