# 首领组件说明

首领组件负责“岁时情劫”玩法。

岁时情劫会在二十四节气和传统节日出现，文案统一走中国风伤痛志怪：年轻男女、旧约、迟信、错过、放下。它不是普通探险怪，也不是异界虫洞，不需要移动到某个地点，出现当天直接挑战。

## 命令

```text
首领
首领状态
挑战首领
首领排行
首领奖励
```

## 出现规则

- 每日 04:00 作为业务日刷新点。
- 当天命中二十四节气或传统节日时，第一次发送首领命令会懒加载生成当日岁时情劫。
- 春节、元宵、端午、七夕、中秋、重阳是高权重传统节日，铭刻之羽更容易掉落。
- 当天没有岁时情劫时，会提示下一次出现时间。

## 挑战规则

- 每人每个首领最多挑战 5 次。
- 每次挑战后冷却 30 分钟。
- 玩家必须处于空闲状态，且血气大于 0。
- 首领血量按服务器玩家生态动态生成，不固定给后期玩家专属。
- 首领被击破或次日 04:00 退去后，参与过的玩家可以领取奖励。

## 铭刻之羽

- 铭刻之羽只能由岁时情劫首领产出。
- 普通探险和异界虫洞不会掉落铭刻之羽。
- 铭刻之羽按实例保存，每一枚都有来源首领、标题和文案。
- 发送 `铭刻之羽` 可以查看自己未使用的遗羽和对应文案。
- 铭刻时可以不指定编号，默认消耗最早获得的一枚。
- 也可以在末尾指定编号，例如：`铭刻武器 武器#12 青云剑 #1`。
- 铭刻成功后，这枚铭刻之羽会直接删除，故事随羽毛一起消散，只留下新的显示名。
